# Checked-in performance baseline

This directory contains the first reproducible SwiftQL performance baseline.
The reports were produced in three independent release-mode processes from the
same commit with 50 warmups and 500 recorded samples per measured case/phase.

The measurements, machine metadata, median spread, material-phase assessment,
and any evidence-backed follow-up issues will be added after the three reports
are recorded. No result in this directory is an absolute CI threshold.
